# AutoScroll
Auto Scroller for use in infinite scroll testing 

# Usage
```javascript
var scroll = new Scroll();
scroll.start({forward: true, speed: 'superfast'});
scroll.stop();
```

# Options
- forward 
  - true: Forward scrolling
  - false: Backward scrolling

- speed
 - slow
 - medium
 - fast
 - superfast
 - extreme
